# 🌤️ Dynamic weather application

A premium, responsive weather application offering an Apple iOS-level user experience. Built with native web technologies to feature dynamic backgrounds, sleek frosted glass cards, and Progressive Web App (PWA) capabilities.

## 🌟 Features
- **Apple iOS Aesthetic:** Stunning frosted glass UI, system typography, and responsive scroll behaviors.
- **Dynamic Backgrounds:** Global visual synchronization adapts live to the local time phase and current weather conditions of any tracked city.
- **Adaptive Dashboards:** Fluidly swaps between a vertical mobile-centric stack to a robust dual-column macOS desktop interface based on screen sizes.
- **PWA Capabilities:** Service Workers equipped for manifest-powered local installation directly onto iOS and Android home screens.
- **Real-time Accuracy:** Backed by the robust OpenWeatherMap API for live and 5-day forecasting.
